## v1.4.8

### 更新亮点

- 新增 `tiktok-ops-assistant` 适配器，支持 TikTok Shop 商品评分抓取、联盟达人视频列表采集与视频下载打包。
- 新增 `tmall-ops-assistant` 适配器，支持按多个天猫商品链接批量抓取买家评价、追评、图片与规格信息。
- TikTok 达人视频下载支持按商品 ID 过滤，导出结果表与视频 ZIP 会统一收口到可见输出目录。
- 后端执行链路补强了多链接解析、逐商品导航执行、CDP `Page.navigate` 复用与早期崩溃状态回收。
- 新增/补强 TikTok 与天猫适配器、入口 URL 选择、下载打包与运行时行为的 Python/Node 回归测试。
